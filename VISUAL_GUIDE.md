## Data Type Visualization

### Molecular Weight
```
INPUT:  5805.5
DISPLAY: 5805.50 g/mol
```

### Purity
```
INPUT:  0.985 (number 0-1)
DISPLAY: 98.50% (percentage badge)
```

### Sequence
```
INPUT:  "GIVEQCCTSICSLYQLENYCN" (string)
DISPLAY: [Scrollable code block with monospace font]
MAX HEIGHT: 128px (overflow-y-auto)
```

### PDB IDs
```
INPUT:  ["1MSO", "1A7F", "4AIY"] (array)
DISPLAY: 
  [1MSO ↗]  [1A7F ↗]  [4AIY ↗]
   ↓         ↓         ↓
   Links to: https://www.rcsb.org/structure/1MSO
```

### Availability Badge
```
"In Stock"  → GREEN background
"Pre-Order" → YELLOW background
"7-10 days" → YELLOW background
(Any other) → YELLOW background
```

## Interaction Patterns

### Collapsible Click
```
User clicks on section header
    ↓
Button toggles state
    ↓
Icon rotates (0° → 180°)
    ↓
Content slides open/closed
    ↓
All in ~200ms smooth animation
```

### Link Click (PDB ID or Supplier URL)
```
User clicks link
    ↓
Opens in new tab/window
    ↓
External link indicator (↗) shown
    ↓
Proper href and rel attributes
```

### Export Click
```
User clicks [PDF/JSON/CSV]
    ↓
File generation in browser
    ↓
Download triggered
    ↓
File name format: synthesis-{topic}-{date}.{ext}
```

## Edge Cases Handled

### Missing Chemical Properties Data
```
┌─────────────────────────────────────────┐
│ [🧬 Chemical Properties] ▼              │
│ ┌─────────────────────────────────────┐ │
│ │ Molecular Weight: [not available]   │ │
│ │ Molecular Formula: [not available]  │ │
│ │ Sequence: [not available]           │ │
│ │ Purity: 98.50%                      │ │
│ └─────────────────────────────────────┘ │
└─────────────────────────────────────────┘
```

### Empty Arrays
```
experimental_methods: []  → Section hidden
pdb_ids: []              → Section hidden
suppliers: []            → Section hidden
suppliers: null          → Section hidden
```

### Long Text Handling
```
Sequence: [████████████████████] ← Scrollable container
                                    max-h-32
                                    overflow-y-auto

```
